
# BankBalance

Built with Next.js, BankBalance is a financial SaaS platform that connects to multiple bank accounts, displays transactions in real-time, and manages their finances altogether.

## Tech Stack

- Next.js
- TypeScript
- Appwrite
- Plaid
- Dwolla
- Zod
- TailwindCSS
- Chart.js
- ShadCN


## Features

- Authentication
- Connect Banks
- Home Page
- My Banks
- Transaction History
- Responsiveness
